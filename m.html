Google Sheets Setup Required

To connect your Google Sheet:

  1. Open your Google Sheet with driver data
  2. Click "Share" and set to "Anyone with the link can view"
  3. Copy the sheet ID from the URL
  4. Update the SHEET_ID variable in the code
  5. Ensure your sheet has headers in the first row

Expected columns: Driver ID, Name, Mobile Number, Email, Status, License Type, etc.

Google Sheets Setup Required

To connect your Track Data Google Sheet:

  1. Open your Google Sheet with track performance data
  2. Click "Share" and set to "Anyone with the link can view"
  3. Copy the sheet ID from the URL
  4. Update the TRACK_SHEET_ID variable in the code
  5. Ensure your sheet has headers in the first row

Expected columns: Track ID, Race Date, Driver Name, Lap Time, Position, Track Name, Weather, etc.
